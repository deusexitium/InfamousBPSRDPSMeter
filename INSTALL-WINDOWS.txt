📦 STEP 1: TRANSFER TO WINDOWS
===============================

From Windows Explorer, navigate to:
  \\wsl$\Ubuntu\development\

Copy this file:
  BPSR-Meter-Enhanced-v2.5.0.tar.gz (11 MB)

Paste to:
  C:\Users\YourName\Desktop\  (or anywhere you prefer)


📂 STEP 2: EXTRACT
===================

Right-click: BPSR-Meter-Enhanced-v2.5.0.tar.gz
Select: "Extract All"
Choose destination
Click: "Extract"

You should now have:
  BPSR-Meter\ folder


🔨 STEP 3: BUILD THE INSTALLER
================================

Open folder: BPSR-Meter\

Right-click: build.bat
Select: "Run as administrator"

Wait 5-10 minutes while it:
  ✓ Checks Node.js
  ✓ Installs pnpm
  ✓ Installs dependencies  
  ✓ Compiles native modules
  ✓ Builds Windows installer

When done, you'll see:
  "✅ BUILD SUCCESSFUL!"


📦 STEP 4: INSTALL NPCAP (Required!)
=====================================

Before running BPSR Meter, install Npcap:

Download from:
  https://npcap.com/

During installation, CHECK these:
  ☑ Install Npcap in WinPcap API-compatible Mode
  ☑ Support loopback traffic

Then RESTART your computer.


🚀 STEP 5: INSTALL BPSR METER
===============================

Navigate to:
  BPSR-Meter\dist_electron\

Right-click: BPSR Meter Setup 2.5.0.exe
Select: "Run as administrator"

Follow the installation wizard.


🎮 STEP 6: RUN & ENJOY
=======================

1. Launch "BPSR Meter" from Start Menu (as Administrator)

2. Start Blue Protocol

3. Enter combat

4. The meter overlay will appear!

5. If no data shows: Change instance/channel once

⚠️  TROUBLESHOOTING
====================

"build.bat hangs or errors":
  → Run as Administrator
  → Install Visual Studio Build Tools
  → Install Python 3.11+

"Application doesn't start":
  → Install Npcap
  → Run as Administrator
  → Check Windows Defender

"No data in overlay":
  → Start game BEFORE meter
  → Change instance once
  → Wait 10 seconds
